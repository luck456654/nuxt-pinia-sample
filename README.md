# nuxt-pinia-sample

## 概要

- [Nuxt3](https://v3.nuxtjs.org/) + [Pinia](https://pinia.vuejs.org/) のサンプルコードです。

## 動作環境

- Node.js - 16.x
- Yarn - 1.22.x

## ライブラリ

- nuxt - 3.0.0-rc.1
- pinia - 2.0.x
- @pinia/nuxt - 0.1.8

## 動作確認

```bash
# パッケージインストール
yarn

# サーバー起動
yarn dev
```

# nuxt-pinia-sample

## Overview

- [Nuxt3](https://v3.nuxtjs.org/) + [Pinia](https://pinia.vuejs.org/) sample code.

## System Requirements

- Node.js - 16.x
- Yarn - 1.22.x

## Library

- nuxt - 3.0.0-rc.1
- pinia - 2.0.x
- @pinia/nuxt - 0.1.8

## Usage

```bash
# package installation 
yarn

# start the server
yarn dev
```
